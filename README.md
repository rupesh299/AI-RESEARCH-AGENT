# AI Research Agent

An intelligent AI-powered research assistant that automates the research process by combining web search and large language models.

The agent accepts a research topic, gathers relevant information from the web using DuckDuckGo Search, analyzes the results with Google's Gemini AI, and generates a well-structured research report with key findings, trends, and conclusions.

## Features

* Automated web research
* AI-powered analysis using Gemini
* Structured report generation
* Streamlit-based user interface
* Free and lightweight technology stack
* Easy deployment to Streamlit Community Cloud

## Tech Stack

* Python
* Google Gemini API
* DuckDuckGo Search
* Streamlit
* Python Dotenv

## How It Works

1. User enters a research topic.
2. The agent searches the web for relevant information.
3. Search results are collected and processed.
4. Gemini AI analyzes the information.
5. A comprehensive research report is generated and displayed.

## Example Queries

* Latest trends in renewable energy
* Applications of quantum computing in healthcare
* Future of autonomous vehicles
* Impact of artificial intelligence on education
